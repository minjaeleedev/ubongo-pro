## 1. UI 구성 요소

### 1.1 메인 메뉴 UI

#### 필수 요소
| 요소 | 설명 | 우선순위 |
|------|------|----------|
| 게임 타이틀 | "Ubongo 3D" 로고 및 타이틀 애니메이션 | 높음 |
| 플레이 버튼 | 싱글플레이 게임 시작 | 높음 |
| 멀티플레이어 버튼 | 멀티플레이 로비 진입 | 높음 |
| 설정 버튼 | 게임 설정 화면 진입 | 중간 |
| 튜토리얼 버튼 | 게임 방법 안내 | 중간 |
| 종료 버튼 | 게임 종료 | 높음 |

#### 추가 요소
- 플레이어 프로필/아바타 표시
- 최고 점수 표시
- 수집한 보석 총계 표시
- 일일 챌린지 배너
- 배경 3D 퍼즐 애니메이션

#### 레이아웃 사양

|                                          |
|         [   플레이   ]                   |
|         [ 멀티플레이 ]                   |
|         [  튜토리얼  ]                   |

+------------------------------------------+
```

### 1.2 게임 플레이 UI (HUD)

#### 상단 영역 (게임 정보)
| 요소 | 위치 | 설명 |
|------|------|------|
| 타이머 | 상단 중앙 | MM:SS 형식, 10초 미만 시 빨간색 깜박임 |
| 점수 | 상단 좌측 | 현재 점수, 변경 시 스케일 애니메이션 |
| 레벨/라운드 | 상단 좌측 | "Round 1/9" 형식 |
| 일시정지 버튼 | 상단 우측 | 게임 일시정지 |
| 보석 카운터 | 상단 우측 | 수집 보석 수 표시 |

#### 하단 영역 (조작 도움말)
| 요소 | 설명 |
|------|------|
| 회전 컨트롤 | Q/E, R, F 키 또는 터치 제스처 아이콘 |
| 힌트 버튼 | 힌트 사용 (보석 소모) |
| 리셋 버튼 | 현재 블록 배치 초기화 |

#### 게임 영역 오버레이
- 현재 선택된 블록 하이라이트
- 유효 배치 위치 표시기
- 2층 높이 가이드라인

#### HUD 레이아웃
```
+------------------------------------------+

|             |           |         | [Gem]|
+------------------------------------------+
|                                          |
|                                          |
|            [게임 플레이 영역]              |
|                                          |
|                                          |
+------------------------------------------+
| [Q][E] Y축회전  [R] X축회전  [F] Z축회전   |

### 1.3 퍼즐 선택 UI

#### 퍼즐 카드 요소
- 퍼즐 미리보기 (3D 회전 가능)
- 난이도 표시 (별 1-5개)
- 목표 시간 표시

## 2. 블록 조작 인터페이스

### 2.1 터치/마우스 드래그 시스템

#### 마우스 조작 (PC)
| 동작 | 기능 |
|------|------|
| 좌클릭 + 드래그 | 블록 선택 및 이동 |
| 좌클릭 (보드 위 블록) | 배치된 블록 해제 |
| 휠 스크롤 | 카메라 줌 인/아웃 |
| 우클릭 + 드래그 | 카메라 회전 |
| 마우스 업 | 블록 배치 시도 |

#### 터치 조작 (모바일/태블릿)
| 제스처 | 기능 |
|--------|------|
| 탭 + 드래그 | 블록 선택 및 이동 |
| 더블 탭 | 배치된 블록 해제 |
| 핀치 | 카메라 줌 인/아웃 |
| 두 손가락 드래그 | 카메라 회전 |
| 두 손가락 회전 | 선택된 블록 Y축 회전 |

#### 드래그 피드백
- 드래그 시작 시 블록 약간 위로 들어올림 (dragHeight = 2f)
- 드래그 중 블록 투명도 80% 적용
- 유효한 위치 위에서 녹색 하이라이트
- 무효한 위치 위에서 빨간색 하이라이트
- 드래그 종료 시 스냅 또는 원위치 복귀

### 2.2 3D 회전 제스처/단축키

#### 키보드 단축키 (현재 구현됨)
| 키 | 기능 | 회전량 |
|----|------|--------|
| Q | Y축 반시계 회전 | -90도 |
| E | Y축 시계 회전 | +90도 |
| R | X축 회전 | +90도 |
| F | Z축 회전 | +90도 |
| ESC | 게임 일시정지 | - |

#### 추가 필요 단축키
| 키 | 기능 |
|----|------|
| Space | 블록 배치 확인 |
| Tab | 다음 블록 선택 |
| Shift+Tab | 이전 블록 선택 |
| Z | 마지막 동작 취소 (Undo) |
| H | 힌트 표시 |

#### 터치 제스처 회전
```
Y축 회전: 블록 위에서 한 손가락 좌/우 스와이프
X축 회전: 블록 위에서 한 손가락 상/하 스와이프 (드래그 아닐 때)
Z축 회전: 두 손가락 회전 제스처

회전 피드백:
- 스와이프 방향 화살표 표시
- 회전 미리보기 고스트 이미지
- 회전 완료 시 스냅 애니메이션
```

### 2.3 블록 선택 피드백

#### 시각적 피드백
| 상태 | 표현 |
|------|------|
| 기본 | 블록 고유 색상 (100% 불투명) |
| 호버 | 외곽선 하이라이트 (흰색 글로우) |
| 선택됨 | 펄스 애니메이션 + 외곽선 강조 |
| 드래그 중 | 투명도 80% + 그림자 분리 |
| 배치됨 | 100% 불투명 + 배치 완료 이펙트 |

#### 오디오 피드백
| 동작 | 사운드 |
|------|--------|
| 블록 선택 | 픽업 사운드 (짧은 클릭) |
| 블록 회전 | 회전 사운드 (휙 소리) |
| 유효 배치 | 성공 사운드 (딱 맞는 소리) |
| 무효 배치 | 실패 사운드 (퉁 소리) |
| 블록 해제 | 해제 사운드 (빠지는 소리) |

#### 햅틱 피드백 (모바일)
| 동작 | 진동 패턴 |
|------|-----------|
| 블록 선택 | 짧은 진동 (50ms) |
| 블록 회전 | 중간 진동 (100ms) |
| 유효 배치 | 강한 진동 (150ms) |
| 무효 배치 | 이중 진동 (50ms + 50ms) |

### 2.4 스냅 그리드 시각화

#### 그리드 표시
```
+---+---+---+---+
| o | o | o | o |  <- 1층 (y=0)
+---+---+---+---+
| o | o | o | o |  <- 2층 (y=1) - 점선 표시
+---+---+---+---+

o = 스냅 포인트 (셀 중앙)
```

#### 시각화 옵션
| 요소 | 설명 |
|------|------|
| 기본 그리드 | 연한 회색 선, 항상 표시 |
| 타겟 영역 | 녹색 반투명 셀 |
| 스냅 가이드 | 블록 드래그 시 가장 가까운 위치 표시 |
| 높이 레벨 | 1층/2층 구분 시각화 (다른 색상) |

#### 레이어 표시
- 1층 (y=0): 진한 불투명 그리드
- 2층 (y=1): 옅은 점선 그리드
- 드래그 시 현재 레이어 강조

## 3. 시각적 피드백

### 3.1 유효한 배치 하이라이트

#### 배치 상태 색상
| 상태 | 색상 | RGB 값 |
|------|------|--------|
| 유효 배치 가능 | 밝은 녹색 | (0.2, 1.0, 0.2, 0.6) |
| 유효하지 않음 | 밝은 빨간색 | (1.0, 0.2, 0.2, 0.6) |
| 타겟 영역 | 연한 녹색 | (0.2, 0.8, 0.2, 0.5) |
| 점유됨 | 회색 | (0.5, 0.5, 0.5, 0.8) |

#### 하이라이트 애니메이션
```csharp
// 유효 배치 시 펄스 효과
유효_색상 -> 밝은_유효_색상 -> 유효_색상 (0.5초 주기)

// 무효 배치 시 흔들림 효과
무효_위치에서 좌우로 5픽셀 흔들림 (0.3초)
```

### 3.2 2층 높이 시각화

#### 레이어 구분
```
    +-----+
   /     /|     <- 2층 (y=1)
  +-----+ |
  |     | +
  |     |/      <- 1층 (y=0)
  +-----+
```

#### 시각화 방법
| 방법 | 설명 |
|------|------|
| 색상 구분 | 1층: 밝은 색, 2층: 약간 어두운 색 |
| 투명도 | 2층 셀은 60% 투명도로 표시 |
| 테두리 | 2층 경계선을 점선으로 표시 |
| 그림자 | 2층 블록이 1층에 그림자 투영 |

#### 높이 인디케이터
- 보드 측면에 "1층/2층" 레이블 표시
- 드래그 중인 블록의 현재 높이 표시
- 블록 그림자로 깊이감 표현

### 3.3 타이머 경고 애니메이션

#### 타이머 상태별 표시 (현재 구현됨)
| 남은 시간 | 색상 | 효과 |
|-----------|------|------|
| 30초 초과 | 흰색 | 없음 |
| 10-30초 | 노란색 | 없음 |
| 10초 미만 | 빨간색 | 깜박임 (0.2초 주기) |

#### 추가 경고 효과
| 남은 시간 | 추가 효과 |
|-----------|-----------|
| 30초 | "30초 남음" 토스트 메시지 |
| 10초 | 화면 테두리 빨간색 글로우 |
| 5초 | 타이머 크기 확대 + 심장박동 애니메이션 |
| 3초 | 카운트다운 사운드 (똑, 똑, 똑) |

#### 긴급 상황 UI
```
+------------------------------------------+
|    [!!! 시간이 얼마 남지 않았습니다! !!!]  |
|        +---------+                       |
|        | 00:05   | <- 크게 확대된 타이머  |
|        +---------+                       |
|              (빨간 테두리 펄스)            |
+------------------------------------------+
```

### 3.4 퍼즐 완료 이펙트

#### 완료 시퀀스
```
1. 마지막 블록 배치 (0ms)
   └─ 배치 사운드 재생

2. 블록 하이라이트 (0-200ms)
   └─ 모든 블록이 순차적으로 밝게 빛남

3. 파티클 이펙트 (200-700ms)
   └─ 반짝이 파티클 방출
   └─ 완료된 퍼즐 중심에서 별 이펙트

4. UI 전환 (700-1200ms)
   └─ "퍼즐 완료!" 텍스트 페이드인
   └─ 점수 카운트업 애니메이션

5. 결과 표시 (1200ms+)
   └─ 별 등급 순차 표시
   └─ 보석 획득 애니메이션
```

#### 파티클 효과
| 파티클 유형 | 설명 |
|-------------|------|
| 별 파티클 | 퍼즐 중앙에서 방사형 확산 |
| 반짝이 | 각 블록에서 무작위 방출 |
| 광선 | 보드에서 위로 뻗어나가는 빛줄기 |
| 원형 웨이브 | 퍼즐에서 퍼져나가는 원형 파동 |

#### 카메라 연출
- 완료 직후 약간 줌인
- 퍼즐을 중심으로 360도 회전 (1회)
- 결과 화면으로 부드럽게 전환

| 정보 | 표시 방법 |
|------|-----------|
| 플레이어 이름 | 프로필 사진 + 닉네임 |
| 진행률 | 배치된 블록 / 총 블록 비율 |
| 상태 | 진행중 / 완료 / 시간초과 |
| 완료 시간 | 완료한 경우 소요 시간 |

#### 실시간 업데이트
- 진행률 바 애니메이션 (부드러운 업데이트)
- 완료 시 팡파레 효과
- 리드 변경 시 알림 (1위 변경)

## 5. 접근성 고려사항

### 5.1 색상 대비

#### WCAG 2.1 AA 기준 준수
| 용도 | 전경색 | 배경색 | 대비율 |
|------|--------|--------|--------|

| 경고 텍스트 | #FFD700 | #2A2A2A | 10.1:1 |
| 오류 텍스트 | #FF6B6B | #2A2A2A | 5.2:1 |

#### 색맹 대응
| 모드 | 변경 사항 |
|------|-----------|
| 일반 | 기본 색상 팔레트 |
| 적록 색맹 | 녹색 -> 청록색, 빨간색 -> 주황색 |
| 청황 색맹 | 파란색 -> 보라색, 노란색 -> 연분홍 |

#### 대안 표시
- 색상만으로 정보 전달하지 않음
- 패턴/아이콘 추가 (유효: 체크마크, 무효: X 표시)
- 텍스트 레이블 병행
